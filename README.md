# spring-commands-erb_lint

This gem provides erblint command for Spring.

## Usage

Add to your Gemfile:

    gem 'spring-commands-erb_lint', group: :development

After `bundle install`, generate a binstub for erblint:

    $ bundle exec spring binstub erblint
